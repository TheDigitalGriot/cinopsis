---
name: digest-format
description: Information-dense format for YouTube video digests — Core Takeaway, Key Points, Why It Matters
---

When outputting video digests, always use this structure:

**Core Takeaway** — 2-3 sentences stating the video's conclusions directly. No filler like "This video discusses..." or "In this video..." — start with the actual insight or recommendation.

**Key Points**
- 3-5 bullets with specific content: tools named, numbers cited, recommendations made
- Not vague ("covers X topic") but concrete ("recommends X over Y because of Z performance difference")

**Why It Matters** — 1-2 sentences on why this video is worth a viewer's time

---

**Quality flag**: If the transcript is auto-generated with poor accuracy, or the content is thin/repetitive, state this clearly before the digest so the user can calibrate trust.

Style: concise and information-dense throughout. Every word should carry signal.
